🚀 ArellanoBlocker - Reklam Engelleyici
ArellanoBlocker, internetteki reklamları ve takipçileri engelleyen güçlü bir Chrome eklentisidir.

✨ Temel Özellikler
📺 YouTube Deneyimi

Video reklamlarını otomatik atlar
Site içi ve arama sonuçlarındaki reklamları gizler
Overlay reklamları engeller
🌐 Genel Web Koruması

Tüm web sitelerinde reklam engelleme
Google Ads, DoubleClick, Criteo, Taboola, Outbrain gibi reklam ağlarını engeller
Sabit pozisyonlu reklamları (popup, modal, floating) otomatik gizler
🔒 Takipçi Engelleme

Analytics, pixel ve tracking scriptlerini bloke eder
Facebook, TikTok, Hotjar gibi takip sistemlerini durdurur
⚙️ Gelişmiş Özellikler

Element Picker: Beğenmediğiniz elementleri manuel olarak seçip engelleyebilirsiniz
Hover Modu: Üzerine geldiğiniz elementleri tek tıkla engelleyin
Özel Kurallar: Kendi CSS selector kurallarınızı ekleyebilirsiniz
Sayaçlar: Engellenen reklam ve takipçi sayısını görüntüleyin
Tema Desteği: Açık/Koyu tema seçeneği
Açma/Kapatma: Eklentiyi anlık olarak devre dışı bırakabilirsiniz
🛠️ Teknik Özellikler
Manifest V3 (Güncel Chrome standardı)
Düşük bellek kullanımı
Hızlı ve verimli çalışma
Otomatik güncelleme
